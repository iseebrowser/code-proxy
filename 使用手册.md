# Code Proxy 使用手册

## 简介

Code Proxy 是一个 Claude Code 代理应用，主要功能：

- **协议转换**：在 OpenAI 和 Anthropic API 之间进行协议转换
- **多供应商支持**：支持切换不同的 AI API 提供商
- **MCP 集成**：通过 MCP（Model Context Protocol）与 Claude Code 交互

## 快速开始

### 1. 启动应用

运行编译好的可执行文件：

```
src-tauri\target\release\code-proxy.exe
```

应用启动后：
- 主窗口显示供应商管理界面
- 系统托盘图标出现（右下角任务栏）

### 2. 配置 MCP（可选）

在 Claude Code 中注册 MCP 服务器，以便通过对话切换供应商：

```bash
claude mcp add --transport http code-proxy http://127.0.0.1:13722
```

注册成功后，可以在 Claude Code 中说：
- "列出所有供应商"
- "切换到供应商 2"

## 供应商管理

### 添加供应商

1. 点击主界面上的"添加"按钮
2. 填写供应商信息：
   - **Provider**：模型名称（如 `qwen3.5-plus`）
   - **Remark**：备注（可选）
   - **API Key**：API 密钥
   - **Base URL**：API 基础地址
   - **API Type**：Anthropic Messages 或 OpenAI Chat Completions
3. 点击"保存"

### 选择供应商

在主界面下拉列表中选择供应商，或在系统托盘菜单中点击供应商名称。

### 删除供应商

选中供应商后，点击删除按钮，确认即可。

## 使用代理

### 启动代理

1. 选择一个供应商
2. 点击"启动代理"按钮

代理启动后：
- 按钮变为"停止代理"
- Claude Code 的 `settings.json` 会自动更新环境变量

### 停止代理

点击"停止代理"按钮即可。

## 系统托盘

应用关闭主窗口后，会最小化到系统托盘。

### 托盘菜单

- **打开主界面**：显示主窗口
- **供应商列表**：当前选中的供应商有勾选标记
- **退出**：完全退出应用

### 托盘操作

- **左键单击**：显示菜单
- **左键双击**：显示主窗口

## 语言设置

应用支持中文和英文两种语言。

### 切换语言

1. 点击工具栏最右侧的 ⚙️ 按钮
2. 在下拉菜单中选择语言：
   - **English**：英文界面
   - **中文**：中文界面

### 语言菜单关闭方式

- 点击菜单外部区域
- 按 ESC 键

### 说明

- 首次启动时，应用会自动检测系统语言并设为默认
- 语言设置会保存到数据库，重启后保持不变
- 鼠标悬停在 ⚙️ 按钮上会显示当前默认语言

## MCP 工具

注册 MCP 服务器后，可使用以下工具：

| 工具 | 说明 | 示例 |
|------|------|------|
| `list_providers` | 列出所有供应商 | "列出所有供应商" |
| `switch_provider` | 切换供应商 | "切换到供应商 2" |
| `get_current_provider` | 获取当前供应商 | "当前使用哪个供应商？" |
| `proxy_status` | 获取代理状态 | "代理运行状态" |

### 通过 MCP 切换供应商

在 Claude Code 对话中直接说：
```
切换到供应商 2
```

切换后：
- 主界面下拉列表自动更新
- 系统托盘菜单自动更新

## 常见问题

### Q: 代理启动失败怎么办？

A: 检查：
1. 供应商的 API Key 是否正确
2. Base URL 是否可访问
3. 网络是否正常

### Q: MCP 工具无法使用怎么办？

A: 检查：
1. Code Proxy 应用是否正在运行
2. MCP 是否已正确注册：`claude mcp list`
3. 端口 13722 是否被占用

### Q: 如何查看代理日志？

A: 代理日志输出到控制台，查看 Code Proxy 运行窗口。

### Q: 如何完全退出应用？

A: 在系统托盘菜单中点击"退出"，或右键托盘图标选择"退出"。
